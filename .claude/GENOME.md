# Genome Privacy Notice

This repo participates in the AGI-1 genome system. The genome is a shared
knowledge base stored at `~/.claude/agi-1-genome/` that helps AI agents
learn from patterns across multiple repos.

## What gets shared
- Error patterns and fixes (anonymized — no code, no secrets)
- Instruction improvements (e.g., "this phrasing prevented a common mistake")
- Anti-patterns (e.g., "this approach consistently fails in Node.js projects")

## What never gets shared
- Source code
- API keys or credentials
- Business logic or proprietary data
- File contents

## Opt out
Delete this file to opt out of genome participation for this repo.

See `~/.claude/skills/agi-1/templates/genome/GENOME.md` for the complete
privacy reference.
