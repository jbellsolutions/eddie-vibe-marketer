# ETHOS.md — Eddie Vibe Marketer

## Mission

Automate the full ad creative lifecycle — from competitor research to content generation to video production to publishing — so a solo founder or small team can run a $300K/yr performance marketing pipeline with one command.

## Principles

1. **Self-improving by default.** Every cycle gets smarter. Winner patterns feed the next generation. Losers get pruned automatically.
2. **Copywriting DNA over generic AI.** Content is powered by 18 legendary copywriter agents (Hormozi, Kennedy, Schwartz, etc.). Never settle for "sounds like ChatGPT."
3. **Personal profiles over business pages.** Real engagement happens on personal accounts. Browser automation posts as the founder, not a brand page.
4. **Cost discipline.** HeyGen ($29/mo) over Arcads ($1,100/mo). Haiku for generation ($0.01/script), Sonnet/Opus only when quality demands it. Every dollar has a job.
5. **Format diversity.** Seven content formats (UGC video, LinkedIn posts, screenshot statics, carousels, b-roll scripts, text overlays, short captions) from a single research cycle.
6. **Crash-safe operations.** Queue saves after every publish. Retries with backoff. No data loss on failure.
7. **Human-in-the-loop where it matters.** Auto-generate everything, but brand voice setup is manual. Quality gate reviews before production. Dry-run before live publishing.

## What This Project Will Never Do

- Post duplicate content to the same platform
- Publish without a quality gate pass
- Store API keys in code or git
- Use "test post" language on live profiles
- Skip root cause investigation when debugging
- Hardcode timezones or platform-specific assumptions
